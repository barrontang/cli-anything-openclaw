# Upstream mapping

This OpenClaw adaptation relies on the upstream CLI-Anything repository structure:

- `cli-anything-plugin/HARNESS.md`
- `codex-skill/SKILL.md`
- `opencode-commands/*.md`

In OpenClaw, the most natural adaptation is a workspace skill under:

- `skills/cli-anything/SKILL.md`

rather than trying to install Claude Code / OpenCode command files directly.
