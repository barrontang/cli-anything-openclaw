# CLI-Anything for OpenClaw

## English
CLI-Anything for OpenClaw adapts the HKUDS CLI-Anything methodology into an OpenClaw-native skill.

Instead of forcing OpenClaw into another tool's plugin model, this repo gives OpenClaw users a cleaner path: use CLI-Anything as a workspace skill.

Repo:
https://github.com/barrontang/cli-anything-openclaw

## 中文
CLI-Anything for OpenClaw 把 HKUDS 的 CLI-Anything 方法论适配成 OpenClaw 原生 Skill。

它不是强行把 OpenClaw 塞进别的插件体系，而是让 OpenClaw 以自己的方式接入 CLI-Anything。

仓库地址：
https://github.com/barrontang/cli-anything-openclaw
