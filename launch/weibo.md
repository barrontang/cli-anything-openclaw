刚刚发布了一个新仓库：**CLI-Anything for OpenClaw**。

这是把 HKUDS 的 CLI-Anything 方法论，适配成 **OpenClaw 原生 Skill** 的一个版本。

它解决的不是“再造一个 CLI-Anything”，而是一个更实际的问题：
**怎样让 OpenClaw 更自然地使用 CLI-Anything 的方法论，去分析软件、生成 agent-friendly CLI、做验证和迭代。**

这个仓库的价值在于：
- 用 OpenClaw 的方式接入 CLI-Anything
- 不强行依赖 Claude Code / OpenCode 的安装路径
- 更适合本地工作区、自动化流程和 agent-native 软件工作流

仓库地址：
https://github.com/barrontang/cli-anything-openclaw

如果你也在做 AI Agent、CLI、自动化工具链，欢迎看看。
