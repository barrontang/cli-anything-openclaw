# Demo: Build Mode with CLI-Anything for OpenClaw

Use this when you want OpenClaw to apply the CLI-Anything methodology to a local project or repository URL and propose or generate an initial harness layout.

## Example prompt

```text
Use cli-anything on ./my-gui-app and build an initial OpenClaw-friendly CLI harness plan.
```

## Expected output

A strong result should include:
- target software summary
- likely command groups
- backend integration notes
- proposed `agent-harness/` structure
- test plan before implementation
