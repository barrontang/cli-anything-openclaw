# Demo: Refine Mode with CLI-Anything for OpenClaw

Use this when a harness already exists and you want OpenClaw to identify missing capabilities or gaps.

## Example prompt

```text
Use cli-anything to refine the harness in ./agent-harness and identify missing commands, tests, and workflow coverage.
```

## Expected output

A strong result should include:
- current harness inventory
- missing command areas
- high-impact additions
- suggested test expansions
- non-breaking improvement path
